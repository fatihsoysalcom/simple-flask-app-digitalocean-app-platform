# Simple Flask App DigitalOcean App Platform

This example demonstrates a basic Python Flask web application. It's designed to be easily deployable on a Platform as a Service (PaaS) like DigitalOcean App Platform, showcasing how developers can focus purely on application code. The app reads its name and port from environment variables, a common pattern in PaaS environments.

## Language

`python`

## How to Run

1. Install Flask: `pip install Flask`
2. Run the application: `python main.py`
3. Open your browser to `http://localhost:8080`
